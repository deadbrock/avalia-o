📱 IPHONE (iOS):
───────────────────────────────────────────────────────────────

1. Abra o app CÂMERA (app nativo do iPhone)
2. Aponte para o QR Code
3. Aguarde 2 segundos
4. Vai aparecer uma NOTIFICAÇÃO AMARELA no topo da tela
5. TOQUE na notificação
6. O Safari vai abrir automaticamente

⚠️ SE NÃO APARECER A NOTIFICAÇÃO:
   - Vá em Ajustes > Câmera
   - Ative "Escanear Códigos QR"

📱 ANDROID:
───────────────────────────────────────────────────────────────

OPÇÃO 1 - Câmera Nativa:
1. Abra o app CÂMERA
2. Aponte para o QR Code
3. TOQUE no banner/popup que aparecer na tela
4. O Chrome vai abrir automaticamente

OPÇÃO 2 - Google Lens (RECOMENDADO):
1. Abra o Google Fotos ou Google
2. Toque no ícone da CÂMERA/LENS
3. Aponte para o QR Code
4. Toque no link
5. Abre automaticamente

OPÇÃO 3 - App de QR Code:
1. Baixe "QR Code Reader" na Play Store
2. Abra o app
3. Aponte para o QR Code
4. Abre automaticamente

💡 DICAS:
───────────────────────────────────────────────────────────────

1. SEMPRE use a câmera NATIVA do celular (não apps de terceiros)
2. O QR Code deve estar BEM ILUMINADO
3. Mantenha o celular a 15-20cm de distância
4. Aguarde 2-3 segundos para o celular reconhecer
5. TOQUE na notificação que aparecer (não espere abrir sozinho)
